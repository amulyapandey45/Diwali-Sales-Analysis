### **Diwali Sales Analysis – Project Explanation**

This project is all about understanding how people shop during the Diwali festival using data. I took a sales dataset and performed step-by-step analysis to find useful insights about customers, their preferences, and which products perform the best.

First, I cleaned the dataset by removing missing values, unnecessary columns, and fixing data types. This step is important because clean data ensures accurate results. After that, I explored the dataset using Python libraries like Pandas for data handling and Matplotlib/Seaborn for creating charts.

I created multiple visualizations to understand different aspects of the data. For example, I analyzed which gender contributes more to sales and found that female customers play a major role. I also looked at age groups and discovered that people between 26–35 years shop the most during Diwali.

Then, I explored state-wise performance to identify which regions generate the highest orders and revenue. I also analyzed customer demographics like marital status and occupation to understand which groups spend more.

Finally, I examined product categories to see what people prefer buying. Categories like Food, Clothing, and Electronics turned out to be the most popular, with Food generating the highest revenue.

Overall, this project shows how data analysis can help businesses understand customer behavior and make better decisions, especially during festive seasons like Diwali.
